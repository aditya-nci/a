Project Overview

This comprehensive project analyzes Walmart Brazil's sales decline using a Kaggle e-commerce dataset. The solution implements a complete data pipeline from cleaning to visualization, incorporating machine learning predictions for delivery optimization and strategic business insights designed to achieve a 10% sales increase.

Key Objectives

  • ML-Powered Predictions: Advanced machine learning model to predict delivery delays, enabling proactive customer service and operational optimization.
  • Strategic Dashboards: Three comprehensive Power BI dashboards covering Sales Analytics, Customer Reviews, and Logistics Operations.
  • Data Pipeline Architecture: Complete ETL pipeline using Python for cleaning, SQL Server for storage, and SSIS for data integration.
  • Business Impact: Strategic insights designed to achieve 10% sales increase through data-driven decision making.

Technical Implementation

clean.ipynb

Data Cleaning Pipeline: Comprehensive Python notebook for data preprocessing, handling missing values, outlier detection, and feature engineering using pandas and NumPy.

To_SQL.ipynb

Data Loading: Automated data transfer from cleaned datasets to SQL Server using SQLAlchemy and optimized bulk insert operations.

pred.ipynb

ML Model Training: Machine learning pipeline using scikit-learn for delivery delay prediction with feature selection and model validation.
