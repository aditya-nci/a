Cross-Platform Film Analytics

Comprehensive analysis of IMDb, Amazon, and Netflix film datasets to uncover trends in ratings, sales, and genre distribution across multiple streaming platforms.

View on GitHub Live Demo

Project Overview

This project analyzes film datasets from three major platforms - IMDb, Amazon Prime Video, and Netflix - to provide comprehensive insights into the entertainment industry. The analysis covers rating trends, genre preferences, content distribution, and platform-specific characteristics.

Key Objectives

  • Compare content libraries across different streaming platforms
  • Analyze rating distributions and user preferences
  • Identify trending genres and content types
  • Examine temporal trends in content production
  • Provide actionable insights for content strategy

Technology Stack

Python Pandas NumPy Matplotlib Seaborn Jupyter Notebook Data Visualization Statistical Analysis

Key Findings

Rating Distribution Patterns

Discovered significant differences in rating distributions across platforms, with Netflix showing higher average ratings for original content compared to licensed content.

Genre Preferences

Drama and Comedy dominate all platforms, but Netflix shows stronger preference for International content while Amazon focuses more on Action and Thriller genres.

Content Production Trends

Analysis revealed a 40% increase in original content production across platforms since 2018, with Netflix leading in international content diversity.

Platform-Specific Insights

IMDb data shows correlation between higher ratings and specific directors/actors, while streaming platforms show seasonal content release patterns.

Data Visualizations

The project includes comprehensive visualizations showing:

  • Rating distribution histograms across platforms
  • Genre popularity heatmaps
  • Temporal trends in content release
  • Platform comparison dashboards
  • Correlation matrices for various metrics

Data Sources & Methodology

Datasets Used

  • IMDb Dataset: Comprehensive movie and TV show ratings, cast, crew information
  • Netflix Dataset: Content catalog with release dates, genres, and descriptions
  • Amazon Prime Dataset: Movie and show listings with ratings and metadata

Analysis Methodology

  • Data cleaning and preprocessing using Pandas
  • Statistical analysis and hypothesis testing
  • Exploratory data analysis (EDA)
  • Visualization creation using Matplotlib and Seaborn
  • Cross-platform comparison and trend analysis
